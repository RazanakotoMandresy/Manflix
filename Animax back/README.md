# backend js -> ts animax

## prerequesite :

- having an runtime javascript, node , deno , bun .
  In my case i use bun
- having mongo db installed and the process mongod running
  create an .env file then put :
  PORT=port you want to listen to
  DATABASE=mongodb://127.0.0.1:27017/<db_name_you_want>

## run the app

just be in the backend folder then type

```bash
$ npm start

```

```bash
$ bun start

```
